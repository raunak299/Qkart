# Qkart

## Fashion Of The Times

An Ecommerce website for brand new clothes

</div>

## **How to install and run locally ?**

```
$ git clone https://github.com/raunak299/Qkart.git
$ cd Qkart/my-app
$ npm install
$ npm start
```

## **Features -**

- Home page
- Product listing page
- Filter and sort products functionalities
- Individual product page
- Cart Managment
- Wishlist managment
- Order management
- Address management
- Checkout page
- Authentication:
  - User Signup
  - User Login
  - logout page
- Fully Responsive

## **Built with -**

- React JS
- React Context API + useReducer
- React Hooks
- React Router
- React Router v5
- Vanilla CSS
- Mockbee Backend
